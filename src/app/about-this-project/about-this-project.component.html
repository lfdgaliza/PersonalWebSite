Web Stack

Of course, the web stack was used here as well =)

Just a little improvement in case of CSS which I made use of SCSS because I would take advantage of some resources from SASS, but without losing compatibility of pure CSS.

Bootstrap

As I am rather a backend than frontend developer, I choose to make use of Bootstrap which gives me a really good grid system and compliance with the most modern patterns of the internet.

Thinking a little about UX, it is good for my visitors to have a well-known and wide-used frontend pattern.
